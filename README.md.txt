# Body na kružnici – webová aplikace (Streamlit)

Tato aplikace umožňuje vykreslit body na kružnici na základě zadaných parametrů:
- souřadnice středu (X, Y)
- poloměr kružnice
- počet bodů
- barva bodů

Výsledný obrázek obsahuje osy se stupnicí a jednotkami (např. v metrech).  
Aplikace také umožňuje export do PDF s parametry úlohy a informacemi o autorovi.
